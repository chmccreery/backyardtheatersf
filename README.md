# backyardtheatersf
Main website for Backyard Theater SF

## Frontend

To add packages:

```
npm install <package>
```

To run the frontend:

```
cd frontend; npm start;
```

## Backend

```
cd backend;
```

To add a dependency:

```
poetry add <dependency>
```

To install the defined dependencies:

```
poetry install
```

To run the backend:

```
poetry run python src/main.py
```
